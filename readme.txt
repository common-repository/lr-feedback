Allows users to submit feedback anywhere on the site using fixed Popup feedback button.

== Description ==

Feedback form is a simple, easy and fully-customizable wordpress plugin used to add popup on fixed position like left side or right side.
You can display any content like contact form, feedback form. Use Shortoce in post/pages/ and any templates file. If you want to show feedback form in all pages then you need to put shortcode in header.php or footer.php
For more features Please contact us at [Logicrays](http://www.logicrays.com/contact-us-logicrays-india/)

== Installation ==

1. Upload Zip file lr-feedback to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Place `<?php echo do_shortcode('[LR_FEEDBACK ] ');?>` in your templates files or You can add [ LR_FEEDBACK ] to post,pages.
